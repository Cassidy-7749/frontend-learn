# 動畫與過渡

CSS 動畫讓網頁元素產生流暢的視覺效果，提升使用者體驗。CSS 提供兩種主要的動畫機制：`transition`（過渡）用於狀態間的簡單轉換，`animation`（動畫）用於更複雜的多步驟動畫。

## Transition（過渡）

`transition` 讓屬性值的改變產生平滑的過渡效果，通常搭配 `:hover`、`:focus` 或 JavaScript 的 class 切換使用。

### 基本語法

```css
.element {
  transition: property duration timing-function delay;
}

/* 範例 */
.button {
  background: #2196F3;
  transition: background 0.3s ease;
}

.button:hover {
  background: #1565C0;
}
```

### 各屬性說明

```css
.element {
  /* 要過渡的屬性（all = 全部） */
  transition-property: background, transform, opacity;

  /* 過渡持續時間 */
  transition-duration: 0.3s;

  /* 時間函數（加速曲線） */
  transition-timing-function: ease;

  /* 延遲時間 */
  transition-delay: 0.1s;
}
```

### 常用 timing-function

| 值 | 效果 |
|----|------|
| `ease` | 預設，慢快慢 |
| `linear` | 等速 |
| `ease-in` | 慢入 |
| `ease-out` | 慢出 |
| `ease-in-out` | 慢入慢出 |
| `cubic-bezier(n,n,n,n)` | 自訂曲線 |

```css
/* 自訂彈跳效果 */
.bounce {
  transition: transform 0.5s cubic-bezier(0.68, -0.55, 0.265, 1.55);
}
```

<Callout type="tip" title="哪些屬性可以 transition？">
只有具有中間值的屬性才能過渡，例如 `color`、`opacity`、`transform`、`width`、`height`、`padding` 等。`display`、`font-family` 這類離散值無法過渡。不過可以使用新的 `transition-behavior: allow-discrete` 來過渡 `display`。
</Callout>

## Transform（變形）

`transform` 讓元素進行 2D 或 3D 變形，不影響文檔流：

```css
.element {
  /* 位移 */
  transform: translate(20px, 10px);
  transform: translateX(50%);

  /* 縮放 */
  transform: scale(1.5);
  transform: scale(1.2, 0.8);

  /* 旋轉 */
  transform: rotate(45deg);

  /* 傾斜 */
  transform: skew(10deg, 5deg);

  /* 組合多個變形 */
  transform: translate(-50%, -50%) scale(1.1) rotate(5deg);
}
```

### 絕對居中技巧

```css
.centered {
  position: absolute;
  top: 50%;
  left: 50%;
  transform: translate(-50%, -50%);
}
```

## @keyframes 與 animation

`@keyframes` 定義動畫的關鍵影格，`animation` 屬性控制動畫的播放：

### 基本語法

```css
@keyframes slideIn {
  from {
    transform: translateX(-100%);
    opacity: 0;
  }
  to {
    transform: translateX(0);
    opacity: 1;
  }
}

.element {
  animation: slideIn 0.5s ease forwards;
}
```

### 多步驟動畫

```css
@keyframes rainbow {
  0%   { background: red; }
  17%  { background: orange; }
  33%  { background: yellow; }
  50%  { background: green; }
  67%  { background: blue; }
  83%  { background: indigo; }
  100% { background: violet; }
}
```

### animation 屬性

```css
.element {
  animation-name: slideIn;
  animation-duration: 0.5s;
  animation-timing-function: ease;
  animation-delay: 0.2s;
  animation-iteration-count: infinite; /* 或數字 */
  animation-direction: alternate;      /* normal | reverse | alternate */
  animation-fill-mode: forwards;       /* none | forwards | backwards | both */
  animation-play-state: running;       /* running | paused */

  /* 縮寫 */
  animation: slideIn 0.5s ease 0.2s infinite alternate forwards;
}
```

<Callout type="warning" title="animation-fill-mode 很重要">
預設情況下，動畫結束後元素會回到初始狀態。使用 `animation-fill-mode: forwards` 讓元素保持在動畫結束時的狀態，`backwards` 讓元素在延遲期間就套用第一個影格的樣式。
</Callout>

## 效能考量

### 高效能屬性

只有以下屬性可以透過 GPU 加速，不會觸發 layout 或 paint：

| 屬性 | 效能 |
|------|------|
| `transform` | 極佳（GPU 合成） |
| `opacity` | 極佳（GPU 合成） |
| `filter` | 良好 |
| `width` / `height` | 差（觸發 reflow） |
| `top` / `left` | 差（觸發 reflow） |
| `margin` / `padding` | 差（觸發 reflow） |

```css
/* 不好：動畫 left 會觸發 reflow */
.bad {
  position: absolute;
  left: 0;
  transition: left 0.3s;
}
.bad:hover { left: 100px; }

/* 好：使用 transform 替代 */
.good {
  transform: translateX(0);
  transition: transform 0.3s;
}
.good:hover { transform: translateX(100px); }
```

### will-change

提前告知瀏覽器某個屬性即將改變，讓瀏覽器預先優化：

```css
.element {
  will-change: transform, opacity;
}
```

<Callout type="warning" title="謹慎使用 will-change">
`will-change` 會佔用額外的記憶體和 GPU 資源。不要在大量元素上使用，也不要設定 `will-change: all`。最佳做法是在動畫開始前添加，結束後移除。
</Callout>

### 尊重使用者偏好

```css
@media (prefers-reduced-motion: reduce) {
  * {
    animation-duration: 0.01ms !important;
    animation-iteration-count: 1 !important;
    transition-duration: 0.01ms !important;
  }
}
```

## 常用動畫技巧

### 交錯動畫（Stagger）

```css
.item {
  animation: fadeIn 0.5s ease forwards;
  opacity: 0;
}
.item:nth-child(1) { animation-delay: 0.1s; }
.item:nth-child(2) { animation-delay: 0.2s; }
.item:nth-child(3) { animation-delay: 0.3s; }
```

### 暫停 / 播放

```css
.animated { animation: spin 2s linear infinite; }
.animated:hover { animation-play-state: paused; }
```
